Приложение отображение карту при помощи leafleat, а также рендерит полигоны из БД
При переходе на localhost:8083/getPolygons будет отображен список полигонов из БД с координатами и цветом
(для начала надо сохранить готовый набор полигонов по localhost:8083/save )
Чтобы открыть карту перейдите на localhost:8083/map,
input поля принимают широту и долготу после нажатия на кнопку submit вас перенесет на место на карте и будут отображены полигоны взятые из БД
